# Dokumentasi Sistem Absensi Magang

## ğŸ“š Daftar Dokumentasi

Dokumentasi sistem absensi magang terbagi menjadi beberapa diagram dan dokumentasi berikut:

### 1. [ERD (Entity Relationship Diagram)](./ERD.md)
Diagram yang menjelaskan struktur database dan relasi antar entitas.
- Entity descriptions
- Relationship details
- Enumerations
- Database constraints

### 2. [UML Class Diagram](./UML-Class-Diagram.md)
Diagram yang menjelaskan struktur class dan model dalam sistem.
- Class structures
- Attributes and methods
- Relationships
- Design patterns

### 3. [Use Case Diagram (UCD)](./Use-Case-Diagram.md)
Diagram yang menjelaskan use case dan aktor dalam sistem.
- Actors (Admin, Peserta Magang, Pembimbing Magang, System)
- Use cases (35 use cases)
- Use case relationships
- Detailed use case descriptions

### 4. [Activity Diagram](./Activity-Diagram.md)
Diagram yang menjelaskan alur aktivitas dalam berbagai proses.
- Login process
- Absensi process
- Pengajuan izin process
- Approval process
- Dashboard process
- Settings management

### 5. [Sequence Diagram](./Sequence-Diagram.md)
Diagram yang menjelaskan interaksi antar komponen dalam sistem.
- Login sequence
- Absensi creation sequence
- Pengajuan izin sequence
- Approval sequence
- Dashboard statistics sequence
- CRUD operations sequences

### 6. [DFD (Data Flow Diagram)](./DFD.md)
Diagram yang menjelaskan alur data dalam sistem.
- Context diagram (Level 0)
- Top level diagram (Level 1)
- Detailed process diagrams (Level 2)
- Data dictionary
- Process descriptions
